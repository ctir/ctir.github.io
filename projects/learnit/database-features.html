Database Features

Scalable
  • Data is still manageable after growth.
Accessible
  • Many users may access the data concurrently.
Accurate
  • Can enforce data constraints.
Secure
  • Access can be controlled and monitored.
Consistent
  • No redundant data. "One version of the truth."
Permanent
  • Changes made to data are never lost.
