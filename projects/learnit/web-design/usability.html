Usability Qualities

Usability.gov talks about 6 important factors to consider when measuring usability:

Intuitive Design
  • How familiar is the site design?
    • Is site navigation familiar/intuitive?
    • is site layout familiar/intuitive?
Ease of Learning
  • How quickly can the user complete tasks?
Efficiency of Use
  • How quickly can the user complete tasks?
Memorability
Upon revisiting the site:
  • Can the user remember how to complete tasks?
  • How long does it take for the user to return to optimal efficiency?
Error Frequency and Severity
  • How often do errors occur?
  • How severe are the errors?
  • How can you recover from the errors?
Subjective Satisfaction
  • Does the user like using the site?
