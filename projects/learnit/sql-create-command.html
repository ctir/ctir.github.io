Creating a table

If you haven't already, create a database to store your tables.

Say we wanted to put a table similar to this in our database:

table_name
attribute1_name DATATYPE PK
attribute2_name DATATYPE
attribute3_name DATATYPE

We would use code like this:

SQL Server Code:
                        CREATE TABLE  table_name(
                            attribute1_name DATATYPE,
                            attribute2_name DATATYPE,
                            attribute3_name DATATYPE
                        );

This code will create a table in your database with your chosen table_name and attribute names with corresponding datatypes, giving us this:

Read more about the CREATE TABLE keyword on docs.microsoft.com.
